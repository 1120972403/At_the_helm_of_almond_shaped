# At_the_helm_of_almond_shaped
# 执掌杏林--校园服务软件，兼容QQ，微信小程序。小程序不断迭代中。
![image](https://github.com/1120972403/At_the_helm_of_almond_shaped/blob/master/images/psb.jpg)
### 支持功能： 支持实名认证 权限设计 校园新闻展示 动态圈 失物招领 闲置物品 政务查询 图书馆业务查询 校园地图 小游戏 中药图谱 校园植物识别等
#### 技术实现： 采用前端部分使用主流方式三端分离、微服务[vue.js]；后端高效分离MVC。前端采用colorUI+自定义css，Api接口采用tp5+restful风格
